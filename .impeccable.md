# Impeccable Design Context — Zen DZ Solutions

## Design Context

### Users
TPE, artisans, auto-entrepreneurs — souvent en reconversion ou en début d'activité. Ils cherchent quelqu'un de compétent ET accessible, pas un cabinet intimidant. Contexte d'utilisation : sur mobile ou PC, souvent en soirée après le travail ou pendant une pause. Niveau de confiance faible envers les experts comptables / juristes « classiques ». Ils veulent qu'on leur parle normalement.

### Two Maquettes — Distinct Identities

#### Maquette 3 — « Chaleur »
**Concept** : Le comptable de quartier qui connaît ton prénom. Warm, organique, humain.
- Tons crème/vert pâle, taches d'encre, blobs animés
- Hero centré avec badge flottant, blobs CSS animés
- Services en bento grid 3 colonnes asymétrique
- "Comment ça marche" : 3 étapes numérotées avec ligne lime
- Stats sur fond sombre avec chiffres en bulles
- Témoignages style chat WhatsApp
- Contact asymétrique chaud
- Fonts : Nunito (body, friendly) + Syne (titres)
- Ton : tu, proche, rassurant, concret

#### Maquette 4 — « Médina »
**Concept** : Cabinet premium qui rappelle l'Algérie sans folklore. Structuré, confiant, élégant.
- Fond clair cassé, géométrie arabesque subtile, noir d'encre et vert algérien
- Hero split : texte à gauche sur fond sombre / visuel arabesque à droite
- Services en grille formelle 2 colonnes avec numéros romains
- "Notre méthode" : timeline verticale
- "Nos engagements" : section statement sur fond vert algérien
- Stats typographiques grands chiffres
- Témoignages en colonnes éditoriales
- Contact formel sobre
- Fonts : Libre Baskerville (titres, authority) + Karla (body, lisible)
- Ton : nous, professionnel, précis, confiant

### Aesthetic Direction
- Pas de purple-to-blue gradients AI-slop
- Pas de glassmorphism générique
- Pas de border-left accent stripes
- Pas de gradient text
- Les deux maquettes partagent les mêmes données mais doivent sembler deux agences différentes
- M3 : lumineuse, douce, accessible → donne envie de téléphoner
- M4 : sobre, architecturale, algéroise → donne confiance avant même de lire

### Design Principles
1. **Chaque maquette doit être mémorable pour une raison différente**
2. **L'Algérie se ressent, ne se clame pas** (géométrie, couleurs, typographie — jamais de drapeau ou de cliché)
3. **Le client TPE ne lit pas, il ressent** — les visuels font le travail, le texte confirme
4. **Accessibilité franco-française** — WCAG AA minimum, police lisible, contrastes corrects
5. **Zéro framework** — HTML/CSS/JS vanilla, performance maximale sur petites connexions
